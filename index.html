▶ YouTube Ad Skipper

YouTubeの広告を自動でスキップ・ミュートするツール集

💻 PC（Windows / Mac）

Chrome拡張機能 PC 無料

Chrome Edge

ブラウザでYouTubeを見るときに広告を自動スキップします。スキップ可能な広告は即スキップ、できない広告はミュート＆16倍速で飛ばします。

  1. 上のボタンからZIPをダウンロード・解凍
  2. Chromeで chrome://extensions/ を開く
  3. 右上の「デベロッパーモード」をオン
  4. 「パッケージ化されていない拡張機能を読み込む」をクリック
  5. 解凍したフォルダを選択してOK

Tampermonkeyスクリプト PC 無料

Chrome Firefox Edge Safari

Tampermonkeyをインストール後、以下のスクリプトを追加するだけで動作します。

  1. ブラウザにTampermonkeyをインストール
  2. Tampermonkeyアイコン →「新規スクリプトを追加」
  3. 下のコードを全部貼り付けてCtrl+Sで保存
// ==UserScript== // @name YouTube Ad Auto Skipper // @namespace http://tampermonkey.net/ // @version 3.0 // @match https://www.youtube.com/* // @grant none // ==/UserScript== (function () { 'use strict'; function tick() { const video = document.querySelector('video'); if (!video) return; const isAd = !!document.querySelector('.ad-showing'); if (isAd) { if (video.duration && video.currentTime < video.duration - 0.1) video.currentTime = video.duration; video.muted = true; const sels = ['.ytp-skip-ad-button','.ytp-ad-skip-button', '.ytp-ad-skip-button-modern','.ytp-ad-skip-button-slot button']; for (const s of sels) { const b = document.querySelector(s); if (b) { b.click(); break; } } for (const b of document.querySelectorAll('button')) { if (b.offsetParent === null) continue; if ((b.textContent||'').match(/スキップ|Skip/)) { b.click(); break; } } } else { if (video.muted) video.muted = false; } } setInterval(tick, 300); })();

📱 Android

YouTubeアドスキッパー APK Android 無料

Android 8.0以上

YouTubeアプリを見ているときに広告のスキップボタンを自動でタップします。アクセシビリティ機能を使用します。

  1. APKをダウンロード
  2. スマホの設定 →「提供元不明のアプリ」を許可
  3. APKファイルをタップしてインストール
  4. アプリを起動して「アクセシビリティ設定を開く」をタップ
  5. 「YouTube Ad Skipper」をオンにする
  6. YouTubeを開いて動画を再生するだけ！

🍎 iPhone

Brave Browser 無料

iOS

iPhoneはAppleの制限によりYouTubeアプリの広告をブロック・スキップすることができません。Brave BrowserでYouTubeを見ることで広告をブロックできます（スキップではなくブロック）。
